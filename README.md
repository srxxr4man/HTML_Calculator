# 🧮 Simple Calculator (HTML & JavaScript)

## 📌 Description
This is a basic **web-based calculator** built using HTML and JavaScript.  
It performs simple arithmetic operations like:
- Addition
- Subtraction
- Multiplication
- Division

The calculator takes two numbers as input and displays the result instantly when a button is clicked.

---

## 🧠 Concepts Used
- 🌐 HTML for structure
- ⚡ Inline JavaScript for functionality
- 🎯 DOM manipulation using `id`
- ➕ Type conversion using unary `+` operator

---

## ⚙️ Features
- Simple and user-friendly interface
- Performs 4 basic arithmetic operations
- Instant result display
- Lightweight (no external libraries)

---

## 🖥️ How It Works

- User enters two numbers
- Clicks an operation button
- JavaScript performs calculation using:
  ```js
  +a.value + +b.value
